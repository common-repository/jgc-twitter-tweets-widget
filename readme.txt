A simple plugin that creates a widget to display your Twitter Timeline.

== Description ==

This plugin creates a widget to display your Twitter Timeline. Just paste your Twitter URL.

You can set theme (light or dark), height and link color.

== Installation ==

For an automatic installation through WordPress:

1. Go to Plugins > Add New
2. Search for 'JGC Twitter Tweets Widget'
3. Click 'Install Now' and activate the plugin
4. Go to Appearance > Widgets

For a manual installation via FTP:

1. Unzip the file 'jgc-twitter-tweets-widget.zip'
2. Upload the 'jgc-twitter-tweets-widget' folder to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Appearance > Widgets

To upload the plugin through WordPress, instead of FTP:

1. Go to Plugins > Add New
2. Click 'Upload Plugin' button
3. Select 'jgc-twitter-tweets-widget.zip' file
4. Click 'Install Now' and activate the plugin
5. Go to Appearance > Widgets
